# 📚 BookShelf - Book Management System

A beautiful, fully-functional React application built for managing a personal library of books. This project was developed as a technical evaluation to demonstrate expertise in modern React development, state management, and real-world REST API integration.

* **Live URL:** [Insert your deployed Vercel/Netlify link here]

## ✨ Key Features
* **Full CRUD Functionality:** Seamlessly View, Add, Edit, and Delete books.
* **REST API Integration:** Connected to a hosted MockAPI endpoint for persistent, cloud-based data storage.
* **Intelligent Filtering:** Real-time search by Title/Author combined with multi-genre filtering.
* **Premium UX:** Features custom loading skeletons, error boundaries, and elegant notification toasts.
* **Cozy Aesthetic:** A refined, "journal-style" UI built with pure CSS, focusing on elegant typography and warm, earthy tones.

## 🛠️ Technical Stack
* **Frontend:** React (Vite)
* **Styling:** Pure CSS (CSS Variables, Flexbox/Grid)
* **Backend API:** MockAPI.io (RESTful JSON API)
* **State Management:** Custom React Hooks (`useBooks`)
* **Icons & Notifications:** Lucide React, React Hot Toast

## 🚀 Local Setup Instructions

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/ishwari03/book-management-system.git

cd book-management-system
```

### 2️⃣ Install Dependencies

```bash
npm install
```

### 3️⃣ Configure Environment Variables

Create a `.env` file in the root directory:

```env
VITE_API_URL=https://YOUR-MOCKAPI-ID.mockapi.io/books
```

Replace `YOUR-MOCKAPI-ID` with your actual MockAPI endpoint.

---

### 4️⃣ Run Development Server

```bash
npm run dev
```

Application will run at:

```bash
http://localhost:5173
```
